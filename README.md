# dynquant
Dynamic quantile models for systemic risk modelling. Implementation contains the following models:

1. CAViaR (different specifications), see [1]. 
2. MVMQCAViaR, see [3].
2. QARDL-MIDAS and QMIDAS, see [2] & [4].
